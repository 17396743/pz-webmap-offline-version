# 制作的原因：

新手入坑僵毁，看攻略后，发现这个在线网站已经没有了，但源文件在github上保存着。
正好群友也没有地图，我也会点编程技术，花了点时间，把这个给封装成exe文件了。
不需要安装环境，直接就可以打开。
虽然会有点bug和少部分功能缺失，但不影响正常使用。```

# 使用教程：
````
使用前请至少用360驱动大师或者其他的方式，补全基础的电脑运行库。自己是百度用的游侠游戏常用运行库安装包v4(700M左右)
